# claude-activity-api

> CLI API for querying and managing Claude activity sessions.

More information: <https://github.com/phillipgreenii/phillipgreenii-nix-support-apps>.

- Check if Claude agents are currently working:

`claude-activity-api is-agent-active`

- List all active Claude sessions as JSON:

`claude-activity-api list`

- Remove stale or orphaned session files:

`claude-activity-api clean`

- Show help information:

`claude-activity-api help`

- Show version information:

`claude-activity-api --version`
